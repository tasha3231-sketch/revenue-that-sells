# revenue-that-sells

Сайт для рекламно-производственного агентства.

## Главная идея
Не показывать хаотичный перечень услуг, а собрать путь клиента: задача → решение → кейс → заявка.

## Стиль
- Тёмная база
- Размытый полупрозрачный фон из кейсов на первом экране
- Акценты: синий + медь
- Дорогой минимализм
- Плавные эффекты, без визуального шума

## Ключевые элементы
- Header с логотипом и hover-эффектом: проходящий градиент по буквам
- Первый экран с сильным оффером
- Квиз-диагностика
- Кейсы через before/after slider
- Аккордеон в кейсах
- FAQ через аккордеон
- Демо-админка + вход владельца
